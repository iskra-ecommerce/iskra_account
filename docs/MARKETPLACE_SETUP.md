# Пошаговая инструкция: OpenCart Marketplace для новичка

## Что такое OpenCart Marketplace?

Это официальный каталог расширений OpenCart. Люди заходят туда, чтобы найти и скачать модули для своего магазина. Если ваше расширение там — его найдут тысячи пользователей.

---

## 1. Регистрация на opencart.com

1. Откройте: https://www.opencart.com/index.php?route=account/register
2. Заполните:
   - **First Name**: ваше имя
   - **Last Name**: ваша фамилия
   - **Email**: ваш email
   - **Password**: придумайте пароль
   - **Country**: ваша страна
3. Поставьте галочку **I have read and agree**
4. Нажмите **Register**
5. Подтвердите email (придёт письмо)

---

## 2. Вход и доступ к разделу расширений

1. Зайдите: https://www.opencart.com/index.php?route=account/login
2. Войдите с вашим email и паролем
3. Нажмите на ваше имя в правом верхнем углу
4. Выберите **Manage Extensions**

---

## 3. Публикация бесплатного расширения

### Шаг 3.1: Подготовка ZIP-файла

Нам не нужно делать это вручную. Когда вы создадите тег `v1.0.0` на GitHub, GitHub Action сам соберет файл:
`iskra_account-v1.0.0.ocmod.zip`

Скачайте его со страницы **Releases** на GitHub:
- Зайдите: https://github.com/iskra-ecommerce/iskra_account/releases
- Нажмите на `v1.0.0`
- Скачайте файл `iskra_account-v1.0.0.ocmod.zip`

### Шаг 3.2: Заполните информацию

В разделе **Manage Extensions** нажмите **Submit Extension** (или **Add New**).

Заполните поля:

| Поле | Что писать |
|------|-----------|
| **Extension Name** | `Iskra Account — Modern Registration & Language Manager` |
| **Extension URL** | `https://github.com/iskra-ecommerce/iskra_account` |
| **Category** | `Modules` |
| **Type** | **Free** (очень важно: выбирайте Free) |
| **Version** | `1.0.0` |
| **Compatible with** | `4.x` |
| **Upload File** | Выберите `iskra_account-v1.0.0.ocmod.zip` |
| **Description (English)** | Краткое описание на английском |
| **Description (Russian)** | Краткое описание на русском |

### Шаг 3.3: Отправьте на модерацию

1. Нажмите **Submit**
2. Ваше расширение отправится на проверку модератору
3. Обычно проверка занимает **1-3 рабочих дня**
4. Если всё хорошо — расширение появится в каталоге
5. Если модератор найдёт ошибки — придёт уведомление с описанием

---

## 4. Что проверяет модератор

Модератор OpenCart проверяет:

- ✅ Файл открывается как `.ocmod.zip` (правильный формат)
- ✅ Есть `ocmod.xml` с уникальным `<code>iskra_account</code>`
- ✅ В XML нет ошибок
- ✅ PHP-код не вызывает ошибок
- ✅ Нет вредоносного кода
- ✅ Нет модификации ядра (мы не трогаем core — это плюс)
- ✅ Описание соответствует функционалу
- ✅ Совместимость с OpenCart 4.x

Наше расширение проходит всё — мы используем только события, не трогаем ядро.

---

## 5. Обновление расширения

Когда вы выпускаете новую версию (например, 1.0.1):

1. Сделайте `git tag v1.0.1` и `git push origin v1.0.1`
2. GitHub Action соберёт новый `.ocmod.zip`
3. Скачайте его со страницы **Releases**
4. Зайдите: **Manage Extensions** → найдите Iskra Account → **Edit**
5. Обновите **Version** на `1.0.1`
6. Загрузите новый `.ocmod.zip`
7. Нажмите **Submit** (модерация снова, но обычно быстрее)

## 6. Частые вопросы

### Сколько стоит публикация бесплатного расширения?
Нисколько. Бесплатно.

### Могу ли я потом сделать расширение платным?
Да. Но если вы начали как бесплатное, лучше оставить бесплатным — репутация важнее.

### Что если модератор отклонил?
Прочитайте причину, исправьте, загрузите снова.

### Нужна ли английская версия описания?
Да, обязательно. Это международная площадка.

### Должен ли я поддерживать расширение?
Желательно. Отвечайте на вопросы пользователей, выпускайте обновления. Это повышает рейтинг и скачивания.
