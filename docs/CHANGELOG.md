# Changelog

## 1.0.0 (2026-06-07)

Initial release.

### Added
- Modern registration form with icon inputs and card layout
- Real-time password strength meter (Weak/Fair/Good/Strong)
- Phone number mask (+7 (XXX) XXX-XX-XX)
- Password visibility toggle (eye icon)
- AJAX email availability check with debounce
- Language preference selector on registration form
- Language cookie management (90 days for guests)
- Language preference saved to `oc_customer.language_preference` (registered users)
- Admin settings page (default language, toggles for features)
- Dark mode support via `prefers-color-scheme`
- Russian and English language packs
- OpenCart event system integration (no core file modifications)
- GitHub Actions release workflow (`.ocmod.zip` build)
