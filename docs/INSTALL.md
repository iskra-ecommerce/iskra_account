# Установка Iskra Account

## Способ 1: Composer (рекомендуется)

```bash
composer require iskra-ecommerce/iskra_account
```

После установки зайдите в админку:
**Расширения → Установщик** → нажмите **Установить** рядом с `Iskra Account`

## Способ 2: OCMod ZIP

1. Скачайте файл `iskra_account-v1.0.0.ocmod.zip` со страницы [Releases](https://github.com/iskra-ecommerce/iskra_account/releases)
2. Зайдите в админку: **Расширения → Установщик**
3. Нажмите **Загрузить** и выберите ZIP-файл
4. Нажмите **Установить**

## Способ 3: Ручная установка

1. Скопируйте папку `extension/iskra_account/` в корень вашего OpenCart
2. Зайдите в админку: **Расширения → Установщик**
3. Найдите `Iskra Account` в списке и нажмите **Установить**

## После установки

1. **Расширения → Модули → Iskra Account** → **Редактировать**
2. Настройте:
   - Язык по умолчанию (первый вход посетителя)
   - Время жизни cookie языка (дней)
   - Вкл/выкл: индикатор пароля, маска телефона, выбор языка при регистрации
3. Нажмите **Сохранить**

## Проверка

Откройте страницу регистрации: `/index.php?route=extension/iskra/account/register`
Форма должна отображаться с новым дизайном, иконками, маской телефона и индикатором пароля.

## Удаление

**Расширения → Установщик** → напротив `Iskra Account` нажмите **Удалить**
