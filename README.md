## Description

Modern registration and language manager for OpenCart 4. Part of the Iskra ecosystem.

**Keywords:** opencart 4 extension, opencart account, opencart registration, opencart language manager, opencart cyrillic, russian opencart.

---

## Features

- Multilingual registration interface
- Real-time field validation
- Phone number masks
- Password strength indicator
- Integration with Iskra Language Pack (RU UK KK BE RO)

## Installation

1. Download `.ocmod.zip` from [GitHub Releases](https://github.com/iskra-ecommerce/iskra_account/releases)
2. Upload via **System → Extensions → Extension Installer**
3. Refresh modification cache
4. Enable module in **Extensions → Modules**

## Compatibility

- OpenCart 4.1+
- PHP 8.1+
